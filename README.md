Add Elasticsearch support to Laravel Scout

## Install

Via Composer

``` bash
$ composer require alhoqbani/laravel-scout-elasticsearch
```

## Usage

``` php
$skeleton = new Alhoqbani\Elastic();
echo $skeleton->echoPhrase('Hello, League!');
```
